## 递归
### 递归代码模板
```java
public void recur(level,param1...){
    if(level>MAX_LEVEL){
        return ;
    }

    process(level);

    recur(level+1,p...);

    //清楚本层的环境
    //如果是回溯需要完成此步，或者判断是否影响了全局变量
}
```
### 递归误区
1.  不要人肉递归
2.  重复问题
3.  数学归纳法
